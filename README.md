# Exploder
Androidデバイスの画面点灯時に、カウントダウンの後にデバイスを爆発させるアプリです。  
⚠️ このアプリの使用は自己責任でお願いします。

## 使い方
1. Exploderアプリで、カウントダウン終了コマンド、タイムリミット、ペナルティを設定する
2. 「設定」アプリ → ユーザー補助 → Exploder → 「Exploderを使用」をONにする

## 機能
- 画面消灯後に再点灯するとカウントダウンが表示される
- カウントダウン表示中にカウントダウン終了コマンドに沿って音量ボタンを押すとカウントダウンが止まりロック画面に戻る
- カウントダウンが0になると画面が暗転し、設定したペナルティ秒数が経過するまで操作できなくなる
- ペナルティ終了後は画面をタップするとロック画面に戻る

## インストール
(Releases)[https://github.com/nos-ice/exploder/releases]からapkをダウンロードしてAndroidデバイスにインストールしてください。
